Qtile window-manager

I’ve been using Qtile for a while now and it’s a fantastic window manager. Just like dwm, Xmonad or Awesome, it organises its windows dynamically keeping them in a stack with predefined layouts so the user can choose the layout he prefers best and let the wm do the dirty work.

Qtile has its source code and config file written in Python, plus a neat documentation comes in support if you fancy to play with it. To have a rough idea on how it works, you can browse through my dotfiles or just take a look at the sample below.
